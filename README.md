# WanAndroid

##1.项目说明

 	此项目为鸿洋大神玩安卓网站（www.wanandroid.com）的安卓端，感谢鸿洋大神提供的API接口，项目采用MVP+Retrofit+RxJava框架，页面风格基于Material Design。
